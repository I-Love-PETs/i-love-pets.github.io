# Healthcare

## Decision Context

Healthcare data is sensitive, regulated, heterogeneous, and often distributed across institutions.

## Useful PETs

Federated learning can support cross-site model training. MPC or PSI can support cohort matching. DP synthetic data can support safer external research artifacts.

## Watch Outs

Rare diseases, small cohorts, and site-specific patterns make output leakage serious. Clinical utility must be validated locally, not only globally.
