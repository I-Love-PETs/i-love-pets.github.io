# Use Cases

Use cases translate PET choices into domain constraints. The same PET can be wise in one domain and irresponsible in another.

| Domain | Typical Need | Likely PETs |
| --- | --- | --- |
| Healthcare | Multi-site research and cohort analytics | FL, DP, MPC, PSI, synthetic data |
| Finance | Fraud, risk, and compliance collaboration | PSI, MPC, TEEs, clean rooms |
| Advertising | Measurement and attribution | Clean rooms, PSI, DP, federated analytics |
| Public sector | Data sharing with public-interest constraints | DP, MPC, synthetic data, clean rooms |
| AI | Training, inference, RAG, and evaluation | FL, DP, TEEs, HE, confidential computing |
