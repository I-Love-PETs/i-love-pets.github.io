# Fix My Itch Template

Use this template for concrete research or engineering problem cards.

```markdown
# Problem Name

## Problem

A specific, concrete problem.

## The Itch

What is frustrating, brittle, expensive, or missing today?

## Why It Matters

Who suffers because this is unsolved?

## Current Workaround

What do people do today?

## Why The Workaround Is Insufficient

Be explicit about the failure.

## What Good Progress Would Look Like

Define success in a way that can be evaluated.

## Difficulty

Good first research problem / medium / hard / moonshot.

## Good For

ML researcher / systems builder / cryptographer / privacy engineer / benchmark maintainer / policy researcher.

## Related PETs

List relevant PETs.

## Possible First Contribution

A small, realistic starting point.
```

## Review Checklist

- Is the problem concrete?
- Is the current workaround described?
- Is success measurable?
- Is there a possible first contribution?
- Is the difficulty level clear?
