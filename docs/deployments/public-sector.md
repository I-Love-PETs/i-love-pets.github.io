# Public Sector Deployments

## Problem

Agencies and public-interest organizations need data collaboration without eroding public trust.

## Architecture

DP public statistics, MPC for interagency analysis, or synthetic data for open-data exploration.

## PETs Used

DP, MPC, synthetic data, clean rooms, and TEEs.

## What Worked

Clear documentation and constrained outputs can improve trust and reviewability.

## Challenges

Procurement, explainability, public accountability, long-term maintenance, and equity impacts.

## Lessons Learned

Technical privacy guarantees must be paired with governance that affected communities can understand.
