# Deployments

Deployment summaries capture what works in real systems, what breaks, and what future builders should check early.

Every deployment summary should include:

- Problem
- Architecture
- PETs used
- What worked
- Challenges
- Lessons learned

## Starter Themes

| Domain | Common Deployment Question |
| --- | --- |
| Healthcare | Can institutions collaborate without pooling patient data? |
| Finance | Can firms compute shared risk signals without revealing customer lists? |
| Advertising | Can measurement happen without exposing raw user-level behavior? |
| Public sector | Can public-interest data sharing preserve trust and accountability? |
