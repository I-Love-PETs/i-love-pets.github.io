# Synthetic Data

## The Itch

Synthetic data is easy to oversell and hard to audit.

## Why It Matters

Bad synthetic data releases can leak private records or mislead downstream users.

## Current State

DP synthetic data is promising, but utility and privacy audits vary widely.

## What Is Unsolved

Privacy auditing, utility measurement, memorization detection, and communication of limitations.

## Possible Directions

Build memorization test suites, benchmark utility across task families, and publish release report templates.

## Difficulty

Beginner to advanced.

## Good For

Data scientists, privacy reviewers, researchers, and open-data teams.

## Related Patterns

DP synthetic data release, synthetic data release pipeline.
