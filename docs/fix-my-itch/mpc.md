# MPC

## The Itch

MPC can protect inputs in joint computation, but developer experience and large-scale deployment remain painful.

## Why It Matters

Organizations will not adopt MPC if every project feels like bespoke cryptographic consulting.

## Current State

Protocols and libraries are improving, but production users still face hard choices around schemas, malicious security, cost, and debugging.

## What Is Unsolved

Better developer abstractions, practical malicious security, deployment runbooks, and large-scale performance benchmarks.

## Possible Directions

Create MPC query templates, protocol-selection guides, failure-mode tests, and benchmark suites for common analytics tasks.

## Difficulty

Intermediate to advanced.

## Good For

Cryptography engineers, platform builders, and data collaboration teams.

## Related Patterns

MPC analytics pipeline, private set intersection, federated analytics.
